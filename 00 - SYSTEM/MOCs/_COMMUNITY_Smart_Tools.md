## Graphify

**Purpose:** Automatically generates knowledge graphs from your vault

**Features:**
- Community detection
- God node identification
- Connection inference
- Interactive visualizations

**Vault Integration:**
- Output: `graphify-out/`
- Graph Report: `GRAPH_REPORT.md`
- Interactive HTML: `graph.html`

---

## MemPalace

**Purpose:** Local AI memory system - no API keys required

**Features:**
- Semantic search across vault
- Memory drawers for organization
- Knowledge graph with timeline
- Agent diary for tracking

**Commands:**
- `/mempalace:init` - Initialize
- `/mempalace:search` - Search memories
- `/mempalace:mine` - Mine content
