## Collaboration with Obsidian (Second Brain)
- **Vault Location**: `E:/Obsidian Vault`  
- **Core Directories**:  
  - `/inbox`: Unprocessed captures – check first.  
  - `/permanent`: Atomic notes – one idea per note, densely linked.  
  - `/wiki`: Auto‑organized knowledge base (concepts, entities, sources) with wikilinks, tags, and hot cache.  
  - `/projects`: Active project folders.  
  - `/logs`: Session logs and MemPalace diary entries.  
  - `/graphify/`: Codebase knowledge graphs – query with `graphify query ""q""`.  
  - `/chats/`: Imported Claude conversations.  
  - `/_templates/`: Note templates.  
- **Key Commands**:  
  - `/wiki` – setup or continue the wiki.  
  - `/save [name]` – file conversation to wiki.  
  - `/ingest [file]` – process source into wiki pages.  
  - `/autoresearch [topic]` – 3‑round web research.  
  - `/canvas` – visual canvas operations.  
  - `lint the wiki` – health check for orphans/dead links.  
  - `graphify update` – refresh code graph after code changes.  
- **Linking Rules**: Use `[[Note Name]]` for wikilinks, tags (`#project`, `#bug`, `#feature`, etc.), and keep one idea per note (Zettelkasten).
