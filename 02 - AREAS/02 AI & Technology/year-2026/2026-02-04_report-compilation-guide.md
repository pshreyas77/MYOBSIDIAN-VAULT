## How to Get a PDF with Pictures:

1. Select all the report text (Ctrl+A)
2. Copy it (Ctrl+C)
3. Open Microsoft Word or Google Docs
4. Paste the content
5. Download the images from the Z.ai image search results that were generated
6. Insert the images into the document
7. Save as PDF

**Option 2: Use Z.ai's Built-in Visual Report**
The "Visual Report" button at the bottom of the chat should generate a formatted version, though the images may still appear as references rather than embedded pictures.

**Option 3: Manual Image Collection**
Since Z.ai searched for and found images for each person during the report generation, you can:
1. Scroll through the chat to find the image search results
2. Save each image individually
3. Compile them into a document with the report text

The complete report content is ready in your Z.ai chat - you just need to choose which method works best for you to compile it into a PDF with actual images.
