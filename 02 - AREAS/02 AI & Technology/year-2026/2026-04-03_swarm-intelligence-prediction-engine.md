### 🧑 You

Build me an application that acts like a "swarm intelligence" prediction engine. The core idea is that I can give it some source material—like a news article, a policy document, or even a chapter from a book—and tell it what I want to predict.

The app should then create a high-fidelity digital sandbox or a parallel world based on my input. It should be populated with thousands of little AI agents, each with its own personality and memory. These agents will interact and evolve, letting me rehearse future scenarios. I want to see a visualization of this simulation and get a detailed prediction report when it's done. It should also have a feature where I can interact with the agents or the final report.

Please build this with a Python backend and a web frontend. It looks like it can be run with Docker, so please set it up that way. Make sure there's an `.env` file where I can add my own API keys for the LLM and Zep Cloud.
