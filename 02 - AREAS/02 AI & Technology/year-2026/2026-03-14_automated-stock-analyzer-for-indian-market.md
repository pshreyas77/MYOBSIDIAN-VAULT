### 🧑 You

Prompt for the AI

You are an expert Python developer and data scientist. Your task is to create a complete, automated stock analyzer for the Indian equity market. The system must:

    Analyse stocks daily that are available on Groww (NSE/BSE listed).

    Use AI/ML techniques to generate a “Buy” or “Hold” recommendation for each stock.

    Send an email every day with a clear analysis report, including which stocks to buy next and which to hold.

Provide the final code, along with clear instructions on how to set it up and run it daily. Follow the step‑by‑step plan below.
Step 1: Define the Scope & Requirements

    The system should cover all major stocks listed on the National Stock Exchange (NSE) and Bombay Stock Exchange (BSE) that are commonly available on Groww. For simplicity, you can start with the NIFTY 50 or NIFTY 500 stocks.

    The output should be a daily email containing:

        A summary table with stock symbols, current price, AI‑generated score, and recommendation (Buy/Hold).

        A short explanation of the top 5 “Buy” recommendations (why the AI chose them).

        Optionally, charts or key technical indicators for those top picks.

    The AI features must include at least one of the following (or a combination):

        A machine learning model (e.g., Random Forest, LSTM) that predicts next‑day price direction based on historical data and technical indicators.

        Fundamental analysis combined with a scoring system (e.g., Piotroski F‑Score, Graham Number) enhanced by ML.

        Sentiment analysis from financial news headlines (using NLP) for each stock.
