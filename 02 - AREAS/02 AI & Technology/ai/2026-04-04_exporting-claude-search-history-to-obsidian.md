## Path 1 — Export All Conversations at Once (Official, Free)

Go to **claude.ai → click your initials (bottom-left) → Settings → Privacy → Export Data**. Claude will email you a download link. The file arrives as a `.dms` file, which is actually a renamed ZIP archive containing your conversations in JSON format.

The catch: JSON isn't Obsidian-friendly out of the box. You'll need to convert it. Here's a quick Python script approach:

```python
import json, os

with open("conversations.json") as f:
    data = json.load(f)

os.makedirs("obsidian_vault/Claude Chats", exist_ok=True)

for convo in data:
    title = convo.get("name", "Untitled")[:50].replace("/", "-")
    filename = f"obsidian_vault/Claude Chats/{title}.md"
    with open(filename, "w") as f:
        f.write(f"# {title}\n\n")
        for msg in convo.get("chat_messages", []):
            role = "**You**" if msg["sender"] == "human" else "**Claude**"
            text = msg.get("text", "")
            f.write(f"{role}:\n{text}\n\n---\n\n")

print("Done!")
```

Drop the resulting `.md` files into your Obsidian vault folder — they'll show up immediately.
