**PROMPT FOR BUILDING A COMPREHENSIVE FICTION DATABASE WEBSITE**

Create a fully functional, visually appealing website that serves as the ultimate database for ALL fictional characters and universes across cartoons, comics, manga, and anime. This should be a Wikipedia-style comprehensive resource with the following specifications:

**CORE FEATURES:**

1. **Complete Character Database**
   - Every character from every series (not just popular ones)
   - Character name, aliases, and alternative names
   - High-quality character images/artwork
   - Detailed character biography and backstory
   - Power levels with quantifiable metrics
   - Abilities, powers, and techniques list
   - Character relationships and affiliations
   - First appearance information
   - Voice actors/creators credits

2. **Universe/Series Information**
   - Complete cosmology breakdown for each fictional universe
   - Power scaling system (universal, multiversal, omniversal, etc.)
   - Story synopsis and major plot points
   - Timeline of events
   - Locations and worlds within the universe
   - Power system explanation (chakra, ki, magic systems, etc.)

3. **Advanced Search & Filter System**
   - Search by character name, series, universe
   - Filter by power level, abilities, character type
   - Filter by medium (cartoon/comic/manga/anime)
   - Filter by publisher/studio (Marvel, DC, Shonen Jump, etc.)
   - Advanced comparison tools between characters

4. **Interactive Features**
   - Character power level comparisons
   - Versus battle simulator with logic-based outcomes
   - User ratings and community discussions
   - Ability to suggest edits/additions (wiki-style)
   - Bookmark favorite characters
   - Create custom character lists

5. **Visual Design Requirements**
   - Modern, dark theme with vibrant accent colors
   - Card-based layout for character browsing
   - Responsive design (mobile, tablet, desktop)
   - Image galleries for each character
   - Interactive power level charts and visualizations
   - Smooth animations and transitions

6. **Database Structure**
   - Organized by: Universe → Series → Characters
   - Cross-referencing between related characters
   - Tagging system for abilities, affiliations, species
   - Version control for character evolution across different media

7. **Content Categories to Include**
   - Western Comics (Marvel, DC, Image, Dark Horse, etc.)
   - Manga/Anime (all genres and publishers)
   - Western Cartoons (Disney, Cartoon Network, Nickelodeon, etc.)
   - Webcomics and webtoons
   - Light novels with adaptations
   - Video game characters from story-driven games

8. **Technical Features**
   - Fast loading times with image optimization
   - API integration for fetching character data
   - Admin panel for content management
   - User authentication system
   - Content moderation tools
   - SEO optimization for discoverability

9. **Unique Features**
   - Power scaling tier system (Street level → Omnipotent)
   - Cosmology complexity ratings
   - "Similar characters" recommendations
   - Character popularity rankings
   - Recent additions/updates feed
   - Community-sourced content with verification

10. **Data to Display for Each Character**
    - Full name and aliases
    - Profile image (multiple angles if available)
    - Series/Universe origin
    - Species/Race
    - Age, height, weight (if applicable)
    - Occupation/Role
    - Affiliations (teams, organizations)
    - Power level (numerical or tier-based)
    - Complete abilities list with descriptions
    - Strengths and weaknesses
    - Notable battles/feats
    - Character development arc
    - Relationships map
    - Quotes
    - Trivia section

**IMPLEMENTATION REQUIREMENTS:**
- Use React for frontend with modern UI components
- Implement infinite scroll for character browsing
- Include image upload and management system
- Create a robust search algorithm
- Build with scalability in mind (potentially millions of characters)
- Include data export/import functionality
- Mobile-first responsive design
- Accessibility features (screen reader compatible)

**STYLE PREFERENCES:**
- Vibrant, comic book-inspired design elements
- Dynamic hover effects on character cards
- Gradient backgrounds and glass-morphism effects
- Clear typography hierarchy
- Interactive tooltips for quick information
- Collapsible sections for detailed information

Please build this as a fully functional prototype with sample data from multiple universes (at least 20-30 characters from different series) to demonstrate all features. Include proper error handling, loading states, and smooth user experience throughout.
