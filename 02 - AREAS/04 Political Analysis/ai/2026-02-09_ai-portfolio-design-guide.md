### 🧑 You

Based on your resume and the desire for a unique yet professional portfolio, here is a tailored prompt to help you build a standout portfolio that reflects your AI/ML and data science focus:

---

**Portfolio Design Prompt:**

Create a clean, modern, and interactive portfolio website that showcases my work in AI, machine learning, and software development. The portfolio should be unique, visually engaging, and professional, with a focus on storytelling through projects.

**Design & Structure:**
- Use a dark/light theme toggle with a sleek, minimalist UI.
- Implement a fixed sidebar or top navigation with: Home, About, Projects, Skills, Achievements, Contact.
- Include a hero section with a short intro, my photo, and tagline: *“Building intelligent systems with data and code.”*
- Use interactive elements like hover effects, smooth scrolling, and animated transitions (but keep it subtle).

**Content Sections:**

1. **Home/Hero:**
   - Name, current role: *“Aspiring Data Scientist & AI Engineer”*
   - Brief mission statement.
   - Call-to-action buttons: *“View Projects”*, *“Download Resume”*

2. **About:**
   - Short bio (2–3 sentences) highlighting my passion for AI/ML and system optimization.
   - Timeline or infographic of my education and key milestones.

3. **Projects:**
   - Each project should have:
     - Project name, tech stack icons, and GitHub/Live demo links.
     - A short description and the problem it solves.
     - Visuals: GIFs, screenshots, or interactive demos (especially for AI agents).
   - Feature **NeoAttend Agent** and **Open Source Agent** prominently with embedded video/walkthrough if possible.

4. **Skills:**
   - Interactive skill matrix or categorized bubbles (Python, TensorFlow, AWS, etc.).
   - Show proficiency levels visually but avoid cliché progress bars.

5. **Achievements & Certifications:**
   - Cards with icons for each achievement/certification.
   - Include Oracle AI Certification and Cloud Computing with badge images if available.

6. **Contact:**
   - Clean contact form.
   - Links to GitHub, LinkedIn, email, and phone.
   - Optional: A small map pin for Hyderabad.

**Technical Requirements:**
- Built with **Next.js** or **Gatsby** for performance.
- Use **Tailwind CSS** or **Styled Components** for styling.
- Ensure full responsiveness and mobile-friendly design.
- Include a blog section placeholder (for future articles on AI/ML).
- Optimize for SEO and fast loading.

**Unique Touches:**
- Add a **“Project Simulator”** where visitors can run a lightweight, interactive version of one of my AI models directly in the browser (using TensorFlow.js or a simple API call).
- Include a **“Tech Stack Journey”** visualization showing how my skills have evolved.
- Implement a **dark/light mode toggle** with a toggle switch that uses AI/ML terminology (e.g., “Training Mode” / “Inference Mode”).
- Add a small **terminal-style component** in the footer that runs a mock command line showing my latest Git commit or a fun AI-generated quote.

**Hosting & Domain:**
- Deploy on **Vercel** or **Netlify**.
- Use a custom domain: `shreyasai.dev` or similar.

**Inspiration References:**
- Look at portfolios of AI researchers on GitHub (e.g., from OpenAI, DeepMind).
- Avoid templates—custom design only.
