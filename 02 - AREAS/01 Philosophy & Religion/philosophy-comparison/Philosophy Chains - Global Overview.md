## ⛓️ Philosophy Traditions Chains

### 1. Greek Philosophy Chain
**Timeline:** 624 BCE – ~200 CE  
**Core Question:** What is reality?

```
PRE-SOCRATIC ERA
Thales → Anaximander → Anaximenes
Pythagoras (independent)
Heraclitus (independent)
Democritus (independent) ← atom theory
Anaxagoras → taught Pericles (politician)

CLASSICAL ERA
SOCRATES (absorbed all above)
  → PLATO
    → ARISTOTLE
        → ALEXANDER THE GREAT

Key lineages:
- Thales → Anaximander → Anaximenes
- Socrates → Plato → Aristotle → Alexander the Great
- Socrates → Antisthenes → Diogenes (Cynic)
```

**Full details:** See [[Shankara_vs_Greek_Philosophers_Showdown]] and [[Shankara_vs_Socrates_Diogenes_Triangular_Showdown]]

### 2. Indian Philosophy Chain
**Timeline:** 1500 BCE – present  
**Core Question:** What is self/liberation?

```
VEDIC ERA (1500–600 BCE)
Vedic Rishis (seers) → Yajnavalkya (~800 BCE) — first named thinker
  → Upanishadic era (800–500 BCE)

REVOLUTIONARY ERA (600–400 BCE)
→ Mahavira (599–527 BCE) — Jainism
→ Buddha (563–483 BCE) — Buddhism
→ Kapila — Samkhya
  → Patanjali — Yoga
→ Kanada — Vaisheshika (atoms!)
→ Badarayana — Brahma Sutras (~200 BCE)
  → Gaudapada
    → Govindapada
      → SHANKARA (788–820 CE) — Advaita Vedanta
        → Ramanuja (1017–1137 CE) — Vishishtadvaita
          → Madhva (1238–1317 CE) — Dvaita

Key lineages:
- Vedic Rishis → Yajnavalkya → Upanishadic scholars
- Buddha → Mahakasyapa → Ananda → [oral tradition] → Nagarjuna
- Shankara's four disciples: Padmapada, Sureshvara, Hastamalaka, Totakacharya
```

### 3. Roman Philosophy Chain
**Timeline:** 100 BCE – 500 CE  
**Core Question:** How to live virtuously?  
*Note: Romans primarily applied Greek philosophy*

```
GREEK STOICS → ROMAN APPLICATION:
Zeno (Greek) → Chrysippus → Posidonius
  → Cicero (translator/bridge)
    → Seneca (advisor to emperor)
      → Musonius Rufus (teacher)
        → Epictetus (slave→teacher)
          → Marcus Aurelius (emperor)

GREEK EPICURUS → Lucretius → Horace → "Carpe Diem" culture
GREEK PLATO → Ammonius Saccas → Plotinus (Neoplatonism)
  → Augustine → Medieval Christianity → Aquinas
ARISTOTLE (Greek) → Boethius (translated) → Medieval Europe
```

**Key figures:**
- Cicero — Translator of Greek philosophy to Latin
- Seneca, Epictetus, Marcus Aurelius — Roman Stoics
- Lucretius — Roman Epicurean poet
- Boethius — Last Roman philosopher, bridge to medieval era

## 💡 Key Insights
1. **Indian tradition** has the longest continuous timeline (3500+ years)
2. **Greek tradition** founded the tools of logic and analysis that shaped Western thought
3. **Islamic tradition** served as the critical bridge preserving and transmitting Greek knowledge to medieval Europe
4. **Chinese tradition** excelled in practical social philosophy that governed East Asia for millennia
5. **Modern philosophy** represents a global synthesis and critique of all previous traditions
