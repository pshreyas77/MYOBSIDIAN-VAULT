## For future Claude

This is **the** decisive paper for the Aryan migration debate. 523 ancient genomes. Establishes that steppe ancestry arrives in South Asia AFTER IVC decline (1200-800 BCE Swat Valley burials). Use this when user needs the primary source for migration theory.

## Key Findings

| Finding | Significance |
|---------|--------------|
| **523 ancient genomes** sequenced | Largest ancient DNA study of South Asia at the time |
| **Zero steppe ancestry in IVC** | Rakhigarhi individual (~2500 BCE) has no steppe input |
| **Steppe ancestry appears 1200-800 BCE** | Swat Valley burials = hard chronological anchor |
| **ANI = IVC + Steppe mixture** | Ancestral North Indians formed from mixture |
| **ASI = IVC + AASI** | Ancestral South Indians = IVC + indigenous hunter-gatherers |
| **10-20% steppe in modern South Asians** | Varies by caste, region, language group |

## Timeline Established

```
2600-1900 BCE: Mature IVC (zero steppe ancestry confirmed)
    ↓
1900-1500 BCE: IVC decline, de-urbanization
    ↓
1500-1200 BCE: Rigveda composition begins (early steppe contact?)
    ↓
1200-800 BCE:  Swat Valley burials show CLEAR steppe ancestry (hard evidence)
    ↓
1900-4200 years ago: ANI-ASI mixture event (Moorjani et al 2013)
```

## Methodology

- Ancient DNA extraction from petrous bone
- Principal component analysis (PCA)
- ADMIXTURE modeling
- f-statistics for ancestry proportions
- D-statistics for population relationships
