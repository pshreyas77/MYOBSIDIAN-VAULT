# Resume Directory

Place your resume PDF file in this directory for the AI agent to analyze.

## Supported Formats
- PDF (text-based, not scanned images)

## Getting a Sample Resume for Testing
If you don't have a resume handy, you can:
1. Create a simple text resume and save it as PDF
2. Use an online resume builder to create a test resume
3. Download a sample resume template from websites like:
   - https://www.resume.com/
   - https://www.canva.com/resumes/
   - https://novoresume.com/

## Naming Convention
You can name your resume file anything descriptive, for example:
- `my_resume.pdf`
- `john_doe_resume.pdf`
- `software_engineer_resume.pdf`

## Testing Without a Real Resume
To test the agent structure without a real resume:
1. Create a simple text file with sample resume content
2. Name it `sample_resume.txt` 
3. The analyzer expects PDF format, so you'll need to convert it or modify the code slightly for testing

For best results, use an actual PDF resume with clear sections for:
- Contact Information
- Professional Summary
- Skills
- Work Experience
- Education
- Certifications (if applicable)
