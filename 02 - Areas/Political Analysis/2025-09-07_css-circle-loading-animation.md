**Main Features:**
- Smooth rotating gradient ring with a tech-inspired blue color scheme

**Included Variants:**
- **Small size** - for inline loading states
- **Fast animation** - for quick operations
- **Glow effect** - red variant with drop-shadow for emphasis
- **Large & slow** - for major page loads

**Key Benefits:**
- Pure CSS - no JavaScript required
- Lightweight and performant
- Easily customizable colors and sizes
- Smooth 60fps animations
- Responsive design

You can easily integrate this into your website by copying the CSS classes. The main loader uses `.circle-loader` and you can combine it with modifier classes like `.loader-small`, `.loader-fast`, etc. The animation works by rotating gradient borders while keeping the inner circle transparent to create the spinning ring effect.
