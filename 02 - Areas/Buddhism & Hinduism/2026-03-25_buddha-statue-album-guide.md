## **📥 HOW TO DOWNLOAD IMAGES LEGALLY:**

### **Method 1: Metropolitan Museum (Open Access)**
```
1. Go to: https://www.metmuseum.org/art/collection/search?q=buddha  
2. Click on any statue image
3. Look for "Public Domain" or "Open Access" label
4. Right-click → Save Image As
5. Image is FREE for personal/educational use
```

### **Method 2: British Museum**
```
1. Go to: https://www.britishmuseum.org/collection/search?keyword=buddha  
2. Click on artifact
3. Look for download icon (↧)
4. Select resolution
5. Free for non-commercial use
```

### **Method 3: Google Arts & Culture**
```
1. Go to: https://artsandculture.google.com/entity/buddha  
2. Click on any image
3. Click three dots (⋮)
4. Select "Download" or "Share"
5. Free for personal use
```

### **Method 4: Wikimedia Commons**
```
1. Go to: https://commons.wikimedia.org/wiki/Category  :Buddha_statues
2. Click on any image
3. Check license (most are Creative Commons)
4. Download button available
5. Free with attribution
```
