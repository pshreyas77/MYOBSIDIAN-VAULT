The system uses SQLite (not MySQL) for portability so no external database setup is required. The dashboard runs at localhost:5000 and includes a live attendance feed page, student management page, analytics page with four charts (trend line, emotion pie, subject bar, engagement heatmap), and an AI chat interface for NLQ queries. All API routes were verified using Flask's test client before packaging.
