Based on the GitHub repository, here's how to launch OpenDia via npx:

## Quick Start with npx

```bash
npx opendia
```

This command will start the OpenDia MCP (Model Context Protocol) server.

## Setup Process

1. **Install the Browser Extension:**
   - Download from the [releases page](https://github.com/aaronjmars/opendia/releases)
   - Go to `chrome://extensions/` (or your browser's extension page)
   - Enable "Developer mode"
   - Click "Load unpacked" and select the extension folder

2. **Configure with Your AI Tool:**

For **Claude Desktop**, add this to your configuration:
```json
{
  "mcpServers": {
    "opendia": {
      "command": "npx",
      "args": ["opendia"]
    }
  }
}
```

For **Cursor** or other AI tools, use the same configuration or follow their specific setup instructions.

## What OpenDia Does

OpenDia lets AI models control your browser automatically. The key advantage? It leverages everything you already have—your logged-in accounts, saved passwords, cookies, wallets, and browsing history. It works with any Chromium-based browser including Chrome, Arc, Edge, Brave, and others.

The tool provides 17 browser automation capabilities including webpage analysis, content extraction, form filling, navigation, and social media posting with anti-detection features.

## Requirements

- Node.js (for npx)
- A Chromium-based browser
- The OpenDia browser extension installed

Once set up, you can ask your AI to perform tasks like summarizing web content, automating social media posts, testing web applications, or managing your browsing workflow.
