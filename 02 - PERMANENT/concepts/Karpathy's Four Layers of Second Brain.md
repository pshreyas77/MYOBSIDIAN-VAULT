# Karpathy's Four Layers of Second Brain

## Explanation
Karpathy's Four Layers of Second Brain framework describes a progressive system for augmenting human cognition with AI:
1. **Knowledge Layer** - Raw information capture and storage (what you know)
2. **Connection Layer** - Linking related ideas across domains (how you connect)
3. **Synthesis Layer** - Creating new understanding from combined knowledge (what you create)
4. **Intelligence Layer** - AI-augmented thinking and decision-making (how you think)

Each layer builds upon the previous one, with the system becoming more intelligent and valuable as it develops through all four layers.

## Why This Matters
This framework provides a roadmap for building effective AI-augmented thinking systems. Most note-taking apps only address the Knowledge Layer, but true cognitive augmentation requires developing all four layers. The obsidian-second-brain implementation specifically targets all four layers through:
- Knowledge Layer: Systematic capture via INBOX, LITERATURE, PERMANENT notes
- Connection Layer: Automatic linking, backlinks, and graph visualization
- Synthesis Layer: Automatic synthesis pages and concept connections
- Intelligence Layer: Six Claude integrations that act as thinking partners

Understanding these layers helps users implement the system correctly and avoid common pitfalls like focusing only on storage without developing the thinking capabilities.
