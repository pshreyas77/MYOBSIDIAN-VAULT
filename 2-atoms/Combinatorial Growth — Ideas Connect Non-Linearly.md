## For future Claude
Combinatorial growth = ideas connecting in non-linear ways to create emergent patterns. This is a core mechanism of how digital gardens generate value beyond individual notes.

## Definition

Combinatorial growth is the phenomenon where ideas in a digital garden connect in non-linear ways, creating emergent patterns that would not exist in isolation. Unlike linear note-taking where each note stands alone, digital gardens leverage interconnection to produce insights that emerge from the structure itself.

## Technical Requirements

For combinatorial growth to occur, a digital garden needs:
- Bidirectional links (backlinks) — see which notes reference this one
- Graph visualization — see the network structure
- Concept-based search — find ideas by topic, not date
- Version history — track how each note evolves
